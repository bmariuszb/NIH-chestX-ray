# NIH ChestX-ray

## Dataset
ChestX-ray14 contains 112,120 X-ray image scans of 30,805 unique individuals with 14 different categories of chest diseases plus one category labeled "No Finding".

Labels
- No Finding
- Hernia
- Emphysema
- Nodule
- Pneumonia
- Consolidation
- Cardiomegaly
- Effusion
- Mass
- Pleural_Thickening
- Atelectasis
- Pneumothorax
- Fibrosis
- Infiltration
- Edema
